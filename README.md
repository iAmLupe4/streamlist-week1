# StreamList – Week 1 

This repository contains the Week 1 StreamList React application developed for EZTechMovie. 
 
## Project Overview 
StreamList is a cloud-based streaming list application that allows users to search for movies or television shows and manage a personalized list of content they want to watch. This project serves as the foundation for a multi-week final project. 
 
## Features 
- React-based single-page application 
- React Router navigation 
- StreamList input logging to the browser console (Week 1 requirement) 
- Placeholder pages for Movies, Cart, and About 
- Custom CSS styling for improved usability 

 ## Technologies Used 
- React 
- React Router DOM 
- JavaScript (ES6) 
- HTML5 
- CSS3 

 ## How to Run the Project 
1. Clone the repository 
2. Run `npm install` 
3. Run `npm start` 
4. Open `http://localhost:3000` in a browser
